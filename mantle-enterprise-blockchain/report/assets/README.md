# Diagram Assets — Mantle Enterprise Blockchain

This directory contains SVG and PNG diagram assets referenced by the research sections (WHI-334 through WHI-398).

Diagrams are named by their source issue and section number (e.g. `WHI-335-s1.1-traditional-vs-canton.svg`). Unnumbered diagrams (`diagram-1` through `diagram-9`, `whi-394-*`) are synthesis-level visuals used in the solution report and presentation.

Files with both `.svg` and `.png` variants (e.g. `WHI-393-modules-panorama`) have the SVG as the canonical source.
